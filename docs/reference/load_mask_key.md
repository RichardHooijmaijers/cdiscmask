# Load a masking key from disk

Load a masking key from disk

## Usage

``` r
load_mask_key(path)
```

## Arguments

- path:

  Path to an `.rds` file previously written by
  [`save_mask_key()`](https://richardhooijmaijers.github.io/cdiscmask/reference/save_mask_key.md).

## Value

A `cdiscmask_key` object.
